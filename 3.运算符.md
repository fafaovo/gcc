### 运算符
    + 算术运算符
        * 除法 
            + 整除整是整数 
            + 小数除整数是小数
        * 取余数 %
            + 只能针对整数
            + 结果看被除数符号
        * if(a = 100)，等同于if(100)
    + 逻辑运算符
        * 逻辑与 && 
            + 从左到右,只要有一个条件为假，后面的表达式不需要判断(执行),整个条件判断为假
        * 逻辑或 ||
            + 从左到右,只要有一个为真,后面的条件不执行，整个条件判断为真
        * 逻辑非 !
            + 逻辑取反
        * 重难点 && 的优先级比 || 高
    + 关系运算符
    + 位运算符
        * 位与 &    
            * 只要有一个为0,结果为0
            * 任何数跟0位与,结果都是0

        * 位或 |    
            * 只要有一个为1,,结果为1

        * 位取反 ~  
            * 0变成1,1变成0

        * 异或 ^    
            * 相同为0,相异为1
            * a ^ a = 0
            * a ^ 0 = a
    + 三目运算符
    + 其他运算符
        * 左移右移 << >>
            + << 左移运算符
                + 数据本身向高位移动，低位补0
                + 左移n位,相当于数据乘以2的n次方
            + >> 右移运算符
                + 算数右移 和 逻辑右移
                + 算数右移:数据本身向低位移动，高位补符号位
                    + 右移n位,相当于除以2的n次方
                + 逻辑右移:数据本身向低位移动，高位补0
                + 不同的操作系统采用的右移方式不同 [算数右移]
        * 前置++和后置++
                + a++ 先使用后++ 
                + ++a 先++后使用
        * i = 1; i+=i++ + ++i; i = 7
        * sizeof() 是个运算符.sizeof内表达式不执行
        * 逗号表达式 变量 = (表达式1,表达式2,表达式3...) 从左到右,最终结果是最右边的结果

### 把一个数第n位二进制置0 1 取反的公式
    + 置0   a&=(~(1<<(n-1)))
    + 置1   a|=(1<<(n-1))
    + 取反  a^=(1<<(n-1))
