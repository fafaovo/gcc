### 数据结构
* 程序 = 算法 + 数据结构
* 时间复杂度:程序的计算执行次数跟问题规模存在数学关系
    + 衡量程序算法的执行效率和次数
    + 表示方法 T(n) = [O(n),O(1)等等]
        + O(1) 循环次数固定的
        + O(n) 循环次数不固定
        + O(n^2) 循环次数是n的平方
    + 规定
        + 取最高次幂,去掉常量 
            + 2n^2+2n+3 O(n^2)
    + 常见写法
        + O(1) -> O(log2n) -> O(n) -> O(n*log2n) ->O(n^2)
            + log2n = 2的多少次方是n
* 空间复杂度:
    + 占用内存空间大小
    + 表示方法 S(n) = O(n)
